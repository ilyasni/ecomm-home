# Architecture Decisions

Здесь фиксируются архитектурные решения, которые влияют на проект в долгую:

- выбор технологий и интеграций;
- изменения в целевой архитектуре;
- критичные компромиссы (производительность, безопасность, стоимость).

Рекомендуемый формат имени файла: `ADR-XXXX-short-title.md`.
